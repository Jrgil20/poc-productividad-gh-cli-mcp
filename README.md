# PoC Productividad - gh CLI + MCP

Este repositorio contiene una prueba de concepto (PoC) para aumentar la productividad
aprovechando las nuevas herramientas y flujos presentados en GitHub Universe 2025.

Por ahora nuestra herramienta principal será la GitHub CLI (`gh`). Aquí validaremos
comandos, scripts y patrones de integración con MCP para mejorar la eficiencia de
operaciones sobre repositorios y flujos de trabajo de desarrollo.

Objetivos:
- Evaluar cómo `gh` acelera tareas comunes del día a día (issues, PRs, releases, etc.).
- Integrar capacidades MCP para orquestación y automatización reproducible.
- Documentar ejemplos y medir mejoras en productividad.

Estado actual:
- Herramienta principal: `gh` (GitHub CLI).
- Fase: inicial — ejemplos y scripts en desarrollo.

Cómo empezar:
1. Instalar la GitHub CLI: https://cli.github.com/
2. Autenticar: `gh auth login`
3. Consultar/ejecutar ejemplos (próximamente) en `./examples`.

Contribuir:
- Abrir issues o PRs proponiendo pruebas, métricas y scripts.

Notas:
- Inspirado por las novedades mostradas en GitHub Universe 2025.
- Se irán añadiendo guías, benchmarks y casos de uso reproducibles.
